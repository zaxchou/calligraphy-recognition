# 中国画题跋区域空间分析工具

基于深度学习的中国画题跋区域定量分析系统，面向清代画家李鱓（1686-1756）绘画作品研究。

## 一、项目背景与研究目标

### 1.1 研究对象
- **画家**：李鱓（1686-1756），清代著名画家
- **研究内容**：绘画作品中书法题跋区域的定量空间分析
- **核心问题**：
  1. 书法题跋在整幅画面中占据多大比例？
  2. 题跋区域与绘画主体区域在画面中如何分布？

### 2.1 功能需求总览

| 需求编号 | 功能名称 | 优先级 | 说明 |
|---------|---------|--------|------|
| 需求1 | 图像上传与展示 | P0 | 支持单张/多张上传，显示名称和尺寸 |
| 需求2 | 自动区域识别与标注 | P0 | 识别题跋/绘画/留白三类区域，叠加半透明色块 |
| 需求3 | 面积比例可视化 | P0 | 饼图 + 百分比数字展示 |
| 需求4 | 空间分布热力图 | P1 | 题跋聚集程度热力图，支持切换查看 |
| 需求5 | 区域边界可视化 | P2 | 线条显示区域接触边界 |
| 需求6 | 多图对比功能 | P0 | 卡片对比视图 + 年代趋势折线图 |

## 四、功能详细说明

### 4.1 需求1：图像上传与展示

**功能描述**：
- 支持拖拽上传和点击上传两种方式
- 支持 jpg/png/bmp 格式
- 支持批量上传多张图像
- 显示图像名称、尺寸（像素）、文件大小

**UI组件**：
- `ImageUploader.vue` - 拖拽/点击上传组件
- `ImageViewer.vue` - 图像展示组件

### 4.2 需求2：自动区域识别与标注

**功能描述**：
- 系统自动识别图像中的三类区域
- 在原图上叠加半透明色块（alpha=0.4）
- 支持用户手动校正区域类别

**简化方案（手动标注）**：
- 用户通过鼠标拖拽矩形框选择区域
- 为每个区域分配类别（题跋/绘画/留白）
- 系统自动计算面积比例

**AI方案（如时间允许）**：
- 使用 SAM (Segment Anything Model) 进行语义分割
- 或使用百度/阿里云图像分割API

**UI组件**：
- `RegionAnnotator.vue` - 区域标注组件
- `RegionEditor.vue` - 区域编辑工具

### 4.3 需求3：面积比例可视化展示

**功能描述**：
- 饼图展示题跋、绘画、留白三者面积占比
- 同时显示百分比数字
- 饼图颜色与区域标注颜色一致

**UI组件**：
- `AreaPieChart.vue` - 面积占比饼图

### 4.4 需求4：空间分布热力图

**功能描述**：
- 将图像划分为 NxN 网格（如 10x10）
- 计算每个网格单元的题跋覆盖率
- 用颜色深浅表示聚集程度（红色=密集，蓝色=稀疏）
- 鼠标悬停显示具体数值

**展示方式**：
- 选项卡切换：原图 / 区域标注图 / 热力图
- 热力图叠加在缩略图上

**UI组件**：
- `HeatmapViewer.vue` - 热力图展示组件
- `ViewSwitcher.vue` - 视图切换选项卡

### 4.5 需求5：区域边界可视化（P2 可选）

**功能描述**：
- 用线条突出显示题跋区域与绘画区域的接触边界
- 不同接触方式用不同颜色表示：
  - 红色线条：题跋与绘画紧密相邻
  - 黄色色块：题跋与绘画重叠（题在画上）

### 4.6 需求6：多图对比功能

**功能描述**：
- 支持同时分析多张图像（建议上限20幅）
- 提供两种对比视图

**视图A - 指标卡片对比**：
- 每张图像生成缩略图卡片
- 卡片显示：图像名称 + 题跋面积占比（大号字体）
- 支持按题跋占比排序

**视图B - 趋势图**：
- 折线图展示题跋占比随年代变化
- X轴：年代（早期/中期/晚期）
- Y轴：题跋面积占比（%）
- 每个数据点可显示作品缩略图预览

**UI组件**：
- `ComparisonCards.vue` - 多图卡片对比
- `TrendChart.vue` - 年代趋势折线图

## 五、数据格式

### 5.1 用户上传数据

**图像文件**：
- 格式：jpg/png/bmp
- 命名：建议使用作品名称

**年代信息（JSON格式）**：
```json
{
  "作品名称": {
    "年代": "1720",
    "时期": "中期",
    "备注": ""
  }
}
```

## 九、备注

1. **中国画特点**：题跋可能与绘画交融（如题写在石头上），AI识别存在误差，需提供手动校正工具
2. **简化方案**：如AI模型开发周期有限，先使用手动框选方案快速实现核心功能
3. **年代排序**：用户可提供年代信息生成趋势图，也可不提供仅做卡片对比
